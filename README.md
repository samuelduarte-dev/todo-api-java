📝 Todo API - Java Spring Boot

A RESTful Task Management API built with Spring Boot and PostgreSQL, designed to create, read, update, and delete tasks efficiently. Developed as part of the Fundamentals of Java Programming course and guided by AI for best practices and implementation.

🚀 Features (Implemented)
Create tasks (POST /tasks)
List all tasks (GET /tasks)
Retrieve a task by ID (GET /tasks/{id})
Delete tasks (DELETE /tasks/{id})
Error handling for tasks that don’t exist

🛠 Technologies
Java 22
Spring Boot 4.0.4
Spring Data JPA
PostgreSQL
Flyway (DB migrations)
Maven

⚡ Quick Start
Configure your PostgreSQL connection in application.properties
Run the Spring Boot application
Test endpoints with Insomnia or Postman
Start managing tasks immediately!

🔮 Future Improvements
